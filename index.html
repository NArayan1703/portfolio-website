FairTicket: Refundable NFT-Based Event Ticketing

A blockchain-based ticketing system built on Solana where event tickets are issued as NFTs. Users can burn their ticket to receive a 70% refund in USDC if they can't attend. Organizers can control resale using advanced token permissions — preventing scalping and ensuring fair pricing for everyone.

Solana Rust NFT USDC Web3
GitHub
How it works
1
Organizer creates event & sets ticket price on Solana
2
User purchases ticket — minted as an NFT to their wallet
3
Can't attend? Burn ticket → receive 70% refund in USDC instantly
